🌐 OILWISE

OILWISE is a full-stack web application that helps users analyze food and oil consumption, provide health insights, and suggest better dietary choices.

---

🚀 Live Demo

🔗 https://oilwise-backend.vercel.app/

---

🧠 Features

🥗 Food & oil consumption analysis

📊 Interactive dashboards and visualizations

🤖 AI-powered recommendations

🗺️ India usage map visualization

🏆 Leaderboard & gamification

🔐 Authentication system
